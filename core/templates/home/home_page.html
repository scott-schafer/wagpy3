{# Ready To Deploy section #}

Ready To Deploy

A better workflow

If you are an individual, small company or a full time web developer, our products will save you time and money! Ready to deploy apps are fully configured and Ready to deploy to a server.

Simply download our code, unzip the files on your local computer and run the deploy script.


If you have coding experience and want to make changes, simply run the application on your local computer, make all the changes you wish, then run the deployment script. The codebase is fully commented as is the user admin panel.

  • Push to deploy. Our products come with a deployment script that makes deployment easy. Simply run the script and it will copy all the files over to your server.
  • SSL certificates. We use Letsencrypt to manage ssl certificates, they are free and come configured with automatic renewal by default. Set it up once and it will continue to renew every few months.
  • Postgres Database. All products run on a Postgres database that is persistant to the local server. Meaning the database is automatically backed up to the server file system in realtime.

We use Postgres databases in our applications. The database runs along side the application container in it's own Docker container and is backed up to the local file system anytime a change is made in the admin dashboard or a CRUD operation is executed anywhere in the application.

If you choose the option to backup your droplet or EC2 instance your database will be included in those backups. So for a very modest fee you can have your database backed as often as you want.

Want a Database in the Cloud? No problem.

Our applications can also be configured for a cloud based database such as a Digitalocean managed database or a AWS Manage Database. Our applications can also be configured to store all of the media assets in the cloud as well. Such as videos, large image files or zipped files using a S3 type of storage such as AWS S3 Storage or Digitalocean Spaces both are a great way to handle all of your madia assets.
